## Prerequisites

- Node.js 16+
- npm or yarn
- Gemini API Key from [Google AI Studio](https://ai.studio)

## Installation

```bash
npm install
```

## Environment Setup

1. Create a `.env.local` file in the root directory
2. Add your Gemini API key:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This app is ready for deployment on:
- Vercel

- Any Node.js hosting
